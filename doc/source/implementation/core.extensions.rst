Extensions to :mod:`scikits.timeseries`
=======================================

:mod:`scikits.hydroclimpy` introduces several functions to manipulate 
:class:`~scikits.timeseries.TimeSeries`.
It also introduces a new subclass of 
:class:`~scikits.timeseries.TimeSeries`, 
:class:`~core.base.ReferencedSeries`.
